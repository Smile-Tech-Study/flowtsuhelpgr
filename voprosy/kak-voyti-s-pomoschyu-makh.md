---
order: 11
title: Как войти с помощью МАХ?
---

Во Flow есть возможность входа через мессенджер MАХ с авторизацией по номеру телефона.

При получении заявки Flow автоматически отправляет на почту гражданину письмо со ссылкой для входа. Однако не у всех граждан есть доступ к своей почте. Поэтому на странице входа гражданин сможет войти в систему через МАХ.

Сделать это можно по кнопке входа через MАХ со страницы ввода почты гражданина для получения ссылки для входа в систему Flow ТГУ **«Войти через Max»**.

При клике на кнопку **«Войти через MАХ»** гражданин перенаправляется в MАХ, где автоматически открывается чат с ботом Flow ТГУ.

В чате бота будет отображаться приветственное сообщение с кнопкой «Поделиться номером». После ее нажатия производится проверка номера, мессенджер MАХ передаёт номер телефона в систему Flow ТГУ. 

:::info 

Важно: номер должен совпадать с указанным на госуслугах. 

:::

В этом случае гражданин получает сообщение:

:::quote 

**Авторизация прошла успешно, войти в личный кабинет (ссылка)**

:::

При переходе по ссылке пользователь автоматически авторизуется в Личном кабинете.

Если же указанный номер не совпадает с тем, который указали при регистрации, то МАХ отправит такое сообщение:

:::quote 

**Пользователь с таким номером телефона не найден.** **Авторизация в личном кабинете может произойти только если ваш текущий номер телефона в MAX совпадает с тем, который подтвержден на портале "Госуслуги".**\
**Для авторизации в личном кабинете, пожалуйста, перейдите по ссылке (ссылка на страницу ввода почты) и попробуйте войти по электронной почте или номеру телефона, подтвержденным на портале** «**Госуслуги".**

:::

При переходе по ссылке пользователь попадет на стандартную страницу ввода почты гражданина для получения ссылки для входа в систему Flow ТГУ.
